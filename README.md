# ImageGallery.Client

### Prerequisites

```
Node v6.10.1
NET Core 2.0.2
VS Code 1.18.0 or VS 2017 15.4.2
```

### Install

```
dotnet restore

cd src\ImageGallery.Client

npm install

dotnet run

http://localhost:5000/home
```
